5# Description
A JAX implementation of Mitotic Stein Variational Gradient Descent. Simple code examples are included in `tests.ipynb`.
Tested on Python 3.10-3.14, JAX 0.9-0.10, optax 0.2.8.

# Installation
To install:
> `pip install  msvgd`

To uninstall:
> `pip uninstall msvgd`

# Dependencies
* `jax`
* `optax`
